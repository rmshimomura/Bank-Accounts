# Bank-Accounts

Bank accounts project using only the terminal

## Node modules used:
- `fs`
- `Chalk (4.1.2)`
- `Inquirer (8.2.0)`

Accounts are stored on files on the `./accounts` directory, on `{account_name}.json` format.

## Available functions: 

- [x] Account creation
- [x] Check current balance
- [x] Deposit
- [x] Withdraw
- [x] Transference between accounts
